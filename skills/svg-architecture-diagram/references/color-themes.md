# 配色主题详细定义

本文件定义了所有可用的配色主题及其完整色值。每个主题包含以下字段：

- `background` — 画布背景色
- `layerFills` — 5 级层级背景色（从上到下依次使用）
- `nodeFill` — 节点填充色
- `nodeStroke` — 节点描边色
- `primaryText` — 主要文字颜色
- `secondaryText` — 次要文字颜色
- `accentColor` — 强调色（侧边栏/高亮）
- `lineColor` — 连接线与箭头颜色

---

## default — 通用多彩

适用场景：通用架构图，产品经理与业务团队沟通

```json
{
  "background": "#FFFFFF",
  "layerFills": ["#FFE0B2", "#FFCC80", "#B9F6CA", "#C5CAE9", "#90CAF9"],
  "nodeFill": "#FFFFFF",
  "nodeStroke": "#333333",
  "primaryText": "#000000",
  "secondaryText": "#333333",
  "accentColor": "#FFAB91",
  "lineColor": "#333333"
}
```
